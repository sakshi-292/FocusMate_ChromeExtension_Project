## <a name="introduction">ğŸ¤– Introduction</a>

**FocusMate** is a productivity-focused Chrome Extension designed to help users stay focused using a **Pomodoro timer**, **active tab time tracking**, and a **distraction blocklist**. It also includes motivational elements like **daily quotes**, **confetti animations**, and **celebration sounds** to make productivity enjoyable.

ğŸ‘‰ **Pomodoro Timer**: 25-minute focus sessions with 5-minute breaks and live countdown.

ğŸ‘‰ **Confetti & Sound Effects**: Celebration animation and sound when a session ends.

ğŸ‘‰ **Distraction Blocklist**: Add/remove distracting websites easily.
